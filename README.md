### Live demo available at [itch.io](https://juninger.itch.io/kirby-clone-demo)

#### Two sets of controls available:

> 1: \
> Move left - a \
> Move right - d \
> Inhale / shoot - 1 (numpad) \
> Jump - spacebar


> 2: \
> Move left - left arrow \
> Move right - right arrow \
> Inhale / shoot - z \
> Jump - spacebar
